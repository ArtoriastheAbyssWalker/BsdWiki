Tecnologías que se deben evitar

El uso de este tipo de tecnologias puede hacer que el MiddleWare no funcione como deberia o simplemente deje de funcionar. Este es un listado de las tecnologías que no son compatibles con MiddleWare:

1.- CCS3: Sweet Alert con CSS3 no funciona en MDW. Evitar EM6 y estilos nuevos de css3 un ejemplo de ello son las variables.

2.- Evitar el data id.

3.- Escribir el id de esta forma ej: < span id="123" data-idmaterial="" >< / span >'.
